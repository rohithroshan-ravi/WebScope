VS Code Web Preview Extension - Example Page

Welcome to WebScope! This example demonstrates the capabilities of the VS Code web preview extension. You can use this file to test the extension's features.

Features

  • Preview any URL directly in VS Code
  • Load local HTML files with file system access
  • Navigate with back button and history tracking
  • Reload button to refresh the preview
  • File dialog to easily select files
  • Secure iframe sandbox environment
  • Maintains panel state when switching tabs

How to Use

  1. Use the toolbar to navigate and control the preview
  2. Enter any URL in the input field and click "Load"
  3. Use the "Back" button to navigate through history
  4. Click "Reload" to refresh the current page
  5. Click "Open File" to select a different HTML file
